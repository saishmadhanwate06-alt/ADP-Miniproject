 🍰 Dessert Paradise – Online Dessert Website

Dessert Paradise is a simple web-based mini project developed using HTML, CSS, and JavaScript.  
It is designed for ordering and exploring delicious desserts through a clean and user-friendly interface.

---

 🚀 Features

- 🏠 Home page with attractive UI
- 🍩 Dessert menu display
- 🛒 Add to Cart functionality
- 📦 Order page
- 💬 Feedback form
- 🔐 Login page (if included)
- 🎨 Responsive design

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Bootstrap (optional)

 ⚙️ Installation & Setup

1️⃣ Download Project
- Download ZIP file or clone repository

---

 2️⃣ Open in VS Code
- Open VS Code
- Click on **File → Open Folder**
- Select project folder

---

 3️⃣ Run Project
- Open `index.html`
- Right click → **Open with Live Server**
  OR
- Directly open in browser

---

 4️⃣ No Backend Required
- This is a **static website project**
- No database or server setup needed
